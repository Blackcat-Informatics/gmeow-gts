# Code de conduite des contributeurs GTS

> Traduction informative de [`CODE_OF_CONDUCT.md`](../../../CODE_OF_CONDUCT.md). Le document anglais demeure la source faisant autorité pour la gouvernance, la sécurité, les versions, les licences, la contribution, les obligations de conduite, les processus de divulgation et les commandes exécutables. Cette traduction suit [`docs/i18n/GLOSSARY.md`](../GLOSSARY.md) et reste informative.

## Objectif

GTS est un projet professionnel open source sous la gérance de Blackcat Informatics® Inc. Nous souhaitons
que la collaboration ici demeure technique, respectueuse et productive.

## Attentes

Nous demandons aux contributeurs de :

- rester techniques et de concentrer les discussions sur le format, les moteurs, la spécification, la documentation, les tests et la conception
- être respectueux et critiquer les idées plutôt que les personnes
- présumer de la bonne intention et poser des questions de clarification avant de faire remonter un problème
- formuler des commentaires précis, exploitables et professionnels
- aider à maintenir un environnement de travail où les contributeurs peuvent réaliser un travail d'ingénierie de haute qualité

## Comportement inacceptable

Ce qui suit n'est pas acceptable dans les espaces du projet tels que les issues, les pull requests, les discussions, les clavardages ou les événements :

- le harcèlement, la discrimination ou les menaces
- les attaques personnelles, l'intimidation ou le harcèlement
- une perturbation soutenue qui fait dérailler la collaboration
- la publication d'informations privées sans consentement
- la participation coordonnée de mauvaise foi, le trollage ou le sealioning

## Application

Les mainteneurs du projet peuvent prendre toute mesure qu'ils jugent appropriée, y compris des avertissements, des demandes de désengagement, la modération des commentaires ou des bannissements des espaces du projet. Les décisions sont définitives.

## Signalement

Si vous subissez ou êtes témoin d'un comportement qui enfreint ce Code de conduite :

- envoyez un courriel à <conduct@blackcatinformatics.ca>
- incluez des liens, le contexte ainsi que toute capture d'écran ou tout journal pertinent
- évitez l'escalade publique pendant qu'un rapport est en cours d'examen

Les rapports sont traités de manière confidentielle et feront l'objet d'un accusé de réception dans un délai de deux jours ouvrables.
