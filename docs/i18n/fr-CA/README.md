# GTS - Graph Transport Substrate

Cette page établit le point d'entrée canadien-français pour la documentation
localisée de GTS.

La version anglaise [`README.md`](../../../README.md) demeure la source
normative pour les règles du protocole, le format de fil, les exemples de
commande, les valeurs lisibles par machine, les politiques de sécurité et les
conditions de licence. Les traductions sont informatives.

Les prochaines tranches de traduction ajouteront les pages localisées dans ce
répertoire en respectant la structure miroir décrite dans
[`docs/i18n/README.md`](../README.md). Les termes doivent suivre le
[`glossaire`](../GLOSSARY.md), et les commandes, chemins, noms de paquet,
identifiants d'API, blocs de code et valeurs de contrat doivent rester
inchangés.
